# Changelog

All notable changes to this project are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-04-30

### Added
- MCP prompts: `research_report`, `quick_note`, `pdf_outline`
- MCP resources: `pdf-it://templates/research-report`, `pdf-it://templates/plain`, `pdf-it://style-guide`, `pdf-it://markdown-cheatsheet`
- Server capabilities now declare `prompts` and `resources` alongside `tools`

### Notes
- Adds genuinely useful primitives for agent workflows, not score-padding
- All three prompts route the agent toward the right `generate_pdf` call shape

## [1.0.0] - 2026-04-29

### Added
- Initial public release
- MCP tools: `generate_pdf`, `list_templates`
- Templates: `research-report` (cover page + auto-generated TOC), `plain` (dense body, no cover)
- Claude Code skill packaged as `SKILL.md` with trigger phrases (`save this as PDF`, `/pdf`, etc.)
- A4 page size, 25/30mm margins, Inter + JetBrains Mono typography
- Page-numbered footer rendered programmatically via pdf-lib
- Cross-platform Chrome detection with `CHROME_PATH` override
- Cross-client compatibility: Claude Desktop, Claude Code, Cursor, Cline, Continue.dev, Zed, Goose, and any MCP-capable client
- Examples folder with a 12-page sample PDF generated by pdf-it itself

### Stack
- TypeScript, MCP SDK, markdown-it, Puppeteer (puppeteer-core), pdf-lib
